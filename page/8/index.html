LeetCode - Fizz Buzz

发表于 2017-07-05 | 分类于 编程

写一个程序，输出从 1 到 n 数字的字符串表示。

  1. 如果 n 是3的倍数，输出“Fizz”；

  2. 如果 n 是5的倍数，输出“Buzz”；

3.如果 n 同时是3和5的倍数，输出 “FizzBuzz”。

示例：

n = 15,

返回:
[
    "1",
    "2",
    "Fizz",
    "4",
    "Buzz",
    "Fizz",
    "7",
    "8",
    "Fizz",
    "Buzz",
    "11",
    "Fizz",
    "13",
    "14",
    "FizzBuzz"
]

思路

  1. 从2开始遍历，将其平方，及平方+v的数，倍数全部置0
  2. 求和

    class Solution:
        def fizzBuzz(self, n):
            """
            :type n: int
            :rtype: List[str]
            """
            tmp = ["" for v in range(n)]
            for v in range(2,n,3):
                tmp[v] += 'Fizz'
            for v in range(4,n,5):
                tmp[v] += 'Buzz'
            for v in range(n):
                if tmp[v] == "":
                    tmp[v] = str(v+1)
            return tmp

LeetCode - 爬楼梯

发表于 2017-07-05 | 分类于 编程

假设你正在爬楼梯。需要 n 步你才能到达楼顶。

每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？

注意：给定 n 是一个正整数。

示例 1：

输入： 2

输出： 2

解释： 有两种方法可以爬到楼顶。

  1. 1 步 + 1 步
  2. 2 步

示例 2：

输入： 3

输出： 3

解释： 有三种方法可以爬到楼顶。

  1. 1 步 + 1 步 + 1 步
  2. 1 步 + 2 步
  3. 2 步 + 1 步

    class Solution:
        def climbStairs(self, n):
            """
            :type n: int
            :rtype: int
            """
            tmp = [0]*(n+1)
            tmp[0] = 1
            tmp[1] = 1
            for i in range(2, n+1):
                tmp[i] = tmp[i-1] + tmp[i-2]
            return tmp[n]

思路1(递推法求解)

  1. 首先求得递推公式 F(n) = F(n-1) + F(n-2)
  2. 实现

    class Solution:
        def get_value(self, n, tmp):
            if n <= 1:
                return 1
            # 如果记忆过，则直接返回值
            if tmp[n] != -1:
                return tmp[n]
            # 否则重新计算
            tmp[n] = self.get_value(n-1, tmp) + self.get_value(n-2, tmp)
            return tmp[n]


        def climbStairs(self, n):
            """
            :type n: int
            :rtype: int
            """
            tmp = [-1]*(n+1)
            return self.get_value(n, tmp)

思路2(记忆性递归)

  1. 递推公式 F(n) = F(n-1) + F(n-2)
  2. 递归实现

            """
            :type n: int

            """
            first = 1

LeetCode - 不同路径

发表于 2017-07-05 | 分类于 编程

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

问总共有多少条不同的路径？

例如，上图是一个7 x 3 的网格。有多少可能的路径？

说明：m 和 n 的值均不超过 100。

示例 1:

输入: m = 3, n = 2

输出: 3

解释:

从左上角开始，总共有 3 条路径可以到达右下角。

  1. 向右 -> 向右 -> 向下

  2. 向右 -> 向下 -> 向右

  3. 向下 -> 向右 -> 向右

示例 2:

输入: m = 7, n = 3

输出: 28

class Solution:
    def uniquePaths(self, m, n):
        """
        :type m: int
        :type n: int
        :rtype: int
        """
        # m是纵，n是横
        # 保持递归中间结果
        tmp = []
        for i in range(n+1):
            tmp.append([-1]*(m+1))
        # 递归调用
        return self.myAns(m, n, tmp)

    def myAns(self, m, n, tmp):
        # 出界认为是0
        if (m <= 0) | (n <= 0):
            return 0
        # 起点
        if (m == 1) & (n == 1):
            return 1
        # 已经递归过直接返回
        if tmp[n][m] != -1:
            return tmp[n][m]
        # 由结果往回走，左边部分，上边部分
        left = self.myAns(m-1, n, tmp)
        up = self.myAns(m, n-1, tmp)
        # 保持结果
        tmp[n][m] = left + up
        # 返回
        return tmp[n][m]

思路1(递归解法)

  1. 从结果回溯，某一点的路径总数等于其上一点，和其左一点的路径数和；
  2. m<= 0 || n<= 0 认为是边界，路径数为0；
  3. 保持递归过程的中间值；

    class Solution:
        def uniquePaths(self, m, n):
            """
            :type m: int
            :type n: int
            :rtype: int
            """
            # 递推解法
            tmp = []
            for i in range(n+1):
                tmp.append([0]*(m+1))
            tmp[1][1] = 1

            for y in range(1, n+1):
                for x in range(1, m+1):
                    if (x==1) & (y==1):
                        continue
                    tmp[y][x] = tmp[y-1][x] + tmp[y][x-1]

            return tmp[n][m]

思路2(递推解法)

  1. 与递归相反，从上到下，递推求解；

LeetCode - 不同路径

发表于 2017-07-05 | 分类于 编程

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

现在考虑网格中有障碍物。那么从左上角到右下角将会有多少条不同的路径？

网格中的障碍物和空位置分别用 1 和 0 来表示。

说明：m 和 n 的值均不超过 100。

示例 1:

输入:

[

[0,0,0],

[0,1,0],

[0,0,0]
]

输出: 2

解释:

3x3 网格的正中间有一个障碍物。

从左上角到右下角一共有 2 条不同的路径：

  1. 向右 -> 向右 -> 向下 -> 向下

  2. 向下 -> 向下 -> 向右 -> 向右


    class Solution:
        def uniquePathsWithObstacles(self, obstacleGrid):
            """
            :type obstacleGrid: List[List[int]]
            :rtype: int
            """
            m = len(obstacleGrid[0])
            n = len(obstacleGrid)

            tmp = []
            for i in range(n+1):
                tmp.append([-1]*(m+1))

            return self.myAns(m, n, tmp, obstacleGrid)

        def myAns(self, m, n, tmp, obstacleGrid):
            if (m <= 0) | (n <= 0):
                return 0
            if (m == 1) & (n == 1):
                return 1 - obstacleGrid[0][0]
            if tmp[n][m] != -1:
                return tmp[n][m]
            # 障碍物的路径数为0
            if obstacleGrid[n-1][m-1] == 1:
                tmp[n][m] = 0
            else:            
                # 左边路径数
                left = self.myAns(m-1, n, tmp, obstacleGrid)
                # 上边路径数
                up = self.myAns(m, n-1, tmp, obstacleGrid)
                tmp[n][m] = left + up

            return tmp[n][m]

思路(递归解法)

  1. 从结果回溯，某一点的路径总数等于其上一点，和其左一点的路径数和；
  2. m<= 0 || n<= 0 认为是边界，路径数为0；障碍点也认为是0；
  3. 保持递归过程的中间值；

LeetCode - 第一个错误的版本

发表于 2017-07-05 | 分类于 编程

你是产品经理，目前正在带领一个团队开发新的产品。不幸的是，你的产品的最新版本没有通过质量检测。由于每个版本都是基于之前的版本开发的，所以错误的版本之后的所有版本都是错的。

假设你有 n 个版本 [1, 2, …, n]，你想找出导致之后所有版本出错的第一个错误的版本。

你可以通过调用 bool isBadVersion(version) 接口来判断版本号 version 是否在单元测试中出错。实现一个函数来查找第一个错误的版本。你应该尽量减少对调用 API 的次数。

示例:

给定 n = 5，并且 version = 4 是第一个错误的版本。

调用 isBadVersion(3) -> false
调用 isBadVersion(5) -> true
调用 isBadVersion(4) -> true

所以，4 是第一个错误的版本。 


    # The isBadVersion API is already defined for you.
    # @param version, an integer
    # @return a bool
    # def isBadVersion(version):

    class Solution(object):
        def firstBadVersion(self, n):
            """
            :type n: int
            :rtype: int
            """
            left = 1; right = n
            while left < right:
                mid = (left + right)//2
                if isBadVersion(mid) == True:
                    right = mid
                else:
                    left = mid + 1
            return left

LeetCode - 寻找峰值

发表于 2017-07-05 | 分类于 编程

峰值元素是指其值大于左右相邻值的元素。

给定一个输入数组 nums，其中 nums[i] ≠ nums[i+1]，找到峰值元素并返回其索引。

数组可能包含多个峰值，在这种情况下，返回任何一个峰值所在位置即可。

你可以假设 nums[-1] = nums[n] = -∞。

示例 1:

输入: nums = [1,2,3,1]
输出: 2
解释: 3 是峰值元素，你的函数应该返回其索引 2。

示例 2:

输入: nums = [1,2,1,3,5,6,4]
输出: 1 或 5 
解释: 你的函数可以返回索引 1，其峰值元素为 2；
     或者返回索引 5， 其峰值元素为 6。

说明:

你的解法应该是 O(logN) 时间复杂度的。

class Solution(object):
    def findPeakElement(self, nums):
        """
        :type nums: List[int]
        :rtype: int
        """
        nums = [-1*float('inf')] + nums + [-1*float('inf')]

        for i in range(1, len(nums)-1):
            if (nums[i]>nums[i-1])&(nums[i]>nums[i+1]):
                return i - 1

        return 0

LeetCode - 缺失数字

发表于 2017-07-05 | 分类于 编程

给定一个包含 0, 1, 2, …, n 中 n 个数的序列，找出 0 .. n 中没有出现在序列中的那个数。

示例 1:

输入: [3,0,1]
输出: 2

示例 2:

输入: [9,6,4,2,3,5,7,0,1]
输出: 8

说明:
你的算法应具有线性时间复杂度。你能否仅使用额外常数空间来实现?


    class Solution(object):
        def missingNumber(self, nums):
            """
            :type nums: List[int]
            :rtype: int
            """
            nums = sorted(nums)
            for i in range(len(nums)-1):
                if nums[i+1] > nums[i] + 1:
                    return i + 1
            if len(nums) == nums[-1]:
                return 0
            else: 
                return len(nums)

LeetCode - 最小路径和

发表于 2017-07-05 | 分类于 编程

给定一个包含非负整数的 m x n 网格，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

说明：每次只能向下或者向右移动一步。

示例:

输入:

[
[1,3,1],

[1,5,1],

[4,2,1]

]

输出: 7

解释: 因为路径 1→3→1→1→1 的总和最小。


    class Solution:
        def minPathSum(self, grid):
            """
            :type grid: List[List[int]]
            :rtype: int
            """

            m = len(grid[0])
            n = len(grid)
            tmp = []
            for i in range(n+1):
                tmp.append([-1]*(m+1))

            return self.myAns(m, n, grid, tmp)

        def myAns(self, m, n, grid, tmp):
            if (m <= 0) | (n <= 0):
                return float('inf') # 最小路径，不应该设置为0， 这里需要特别注意
            if (m == 1)&(n == 1):
                return grid[0][0]
            if tmp[n][m] != -1:
                return tmp[n][m]

            tmp[n][m] = min(self.myAns(m-1, n, grid, tmp), self.myAns(m, n-1, grid, tmp)) + grid[n-1][m-1]            
            return tmp[n][m]

思路1(递归解法)

  1. 从右下角往回递归，分别比较上方和左方的路径和大小，选择小的加上本点的大小；
  2. 保持递归过程中的值

    class Solution:
        def minPathSum(self, grid):
            """
            :type grid: List[List[int]]
            :rtype: int
            """
            m = len(grid)
            n = len(grid[0])
            for i in range(m):# 多少列 横轴
                for j in range(n):# 多少行 纵轴
                    if (i==0)&(j==0):
                        continue
                    elif i == 0:
                        grid[i][j] += grid[i][j-1]
                    elif j == 0:
                        grid[i][j] += grid[i-1][j]
                    else:
                        grid[i][j] += min(grid[i-1][j], grid[i][j-1])
            return grid[m-1][n-1]

 输出: 3

 rorse -> rose (删除 'r')
 rose -> ros (删除 'e')

    示例 2:

    输入: word1 = "intention", word2 = "execution"
    输出: 5
    解释: 
    intention -> inention (删除 't')
    inention -> enention (将 'i' 替换为 'e')
    enention -> exention (将 'n' 替换为 'x')
    exention -> exection (将 'n' 替换为 'c')

    class Solution:
        def minDistance(self, word1, word2):

            :type word1: str
            :type word2: str
            :rtype: int

        def myMethod(self, word1, word2, l1, l2, tmp):

LeetCode - 买卖股票的最佳时机

发表于 2017-07-05 | 分类于 编程

给定一个数组，它的第 i 个元素是一支给定股票第 i 天的价格。

如果你最多只允许完成一笔交易（即买入和卖出一支股票），设计一个算法来计算你所能获取的最大利润。

注意你不能在买入股票前卖出股票。

示例 1:

输入: [7,1,5,3,6,4]
输出: 5
解释: 在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格。

示例 2:

输入: [7,6,4,3,1]
输出: 0
解释: 在这种情况下, 没有交易完成, 所以最大利润为 0。

思路

  1. 遍历过程中记录最低股价；
  2. 同时记录最高差价；

    class Solution:
        def maxProfit(self, prices):
            """
            :type prices: List[int]
            :rtype: int
            """
            res = 0
            buy = float('inf')
            for v in prices:
                buy = min(buy, v)
                res = max(res, v-buy)
            return res
