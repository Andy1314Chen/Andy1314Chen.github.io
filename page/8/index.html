数值的整数次方(剑指offer第十二题)

发表于 2017-09-01 | 分类于 数据结构

题目描述

给定一个double类型的浮点数base和int类型的整数exponent。求base的exponent次方。

代码


    # -*- coding:utf-8 -*-
    class Solution:
        def Power(self, base, exponent):
            # write code here
            res = base
            if exponent == 0:
                return 1
            elif exponent > 0:
                flag = 0
            else:
                flag = 1
            exponent = abs(exponent)
            while exponent > 1:
                res *= base
                exponent -= 1
            if flag:
                return 1./res
            else:
                return res

旋转数组的最小数字(剑指offer第六题)

发表于 2017-09-01 | 分类于 数据结构

题目描述

把一个数组最开始的若干个元素搬到数组的末尾，我们称之为数组的旋转。 输入一个非减排序的数组的一个旋转，输出旋转数组的最小元素。 例如数组{3,4,5,1,2}为{1,2,3,4,5}的一个旋转，该数组的最小值为1。 NOTE：给出的所有元素都大于0，若数组大小为0，请返回0。

代码


    # -*- coding:utf-8 -*-
    class Solution:
        def minNumberInRotateArray(self, rotateArray):
            # write code here
            if len(rotateArray) < 1:
                return 0
            i = 1
            while i < len(rotateArray):
                if rotateArray[i] < rotateArray[i-1]:
                    return rotateArray[i]
                i += 1
            return rotateArray[-1]

调整数组顺序使奇数位于偶数前面(剑指offer第十三题)

发表于 2017-09-01 | 分类于 数据结构

题目描述

输入一个整数数组，实现一个函数来调整该数组中数字的顺序，使得所有的奇数位于数组的前半部分，所有的偶数位于数组的后半部分，并保证奇数和奇数，偶数和偶数之间的相对位置不变。

代码


    # -*- coding:utf-8 -*-
    class Solution:
        def reOrderArray(self, array):
            # write code here
            if len(array) < 1:
                return []
            array1 = []
            array2 = []
            for v in array:
                if v%2:
                    array1.append(v)
                else:
                    array2.append(v)
            array = array1 + array2
            return array

青蛙台阶(剑指offer第八题)

代码


    # -*- coding:utf-8 -*-
    class Solution:
        def jumpFloor(self, number):
            # write code here

            first = 1
            second = 1
            res = 1
            for i in range(1, number):
                res = first + second
                second = first
                first = res
            return res

二维数组中的查找(剑指offer第一题)

发表于 2017-09-01 | 分类于 数据结构

题目描述

在一个二维数组中（每个一维数组的长度相同），每一行都按照从左到右递增的顺序排序，每一列都按照从上到下递增的顺序排序。请完成一个函数，输入这样的一个二维数组和一个整数，判断数组中是否含有该整数。

代码1


    # -*- coding:utf-8 -*- 暴力解法
    class Solution:
        # array 二维列表
        def Find(self, target, array):
            # write code here
            n = len(array)
            m = len(array[0])
            for i in range(n):
                for j in range(m):
                    if array[i][j] == target:
                        return True
            return False

代码2


    # -*- coding:utf-8 -*-  每次查找右上角的值的大小
    class Solution:
        # array 二维列表
        def Find(self, target, array):
            # write code here
            col = len(array[0])
            row = len(array)
            cur_col = col - 1
            cur_row = 0
            while (cur_row<row)&(cur_col>=0):
                if array[cur_row][cur_col] == target:
                    return True
                elif array[cur_row][cur_col] > target: # 排除该列
                    cur_col -= 1
                elif array[cur_row][cur_col] < target: # 排除该行
                    cur_row += 1
            return False

链表中倒数第K个节点(剑指offer第十四题)

发表于 2017-09-01 | 分类于 数据结构

题目描述

输入一个链表，输出该链表中倒数第k个结点。

代码


    # -*- coding:utf-8 -*-
    # class ListNode:
    #     def __init__(self, x):
    #         self.val = x
    #         self.next = None

    class Solution:
        def FindKthToTail(self, head, k):
            # write code here
            # 快慢指针，快指针先走k步
            if k == 0:
                return None

            cnt = 1
            first = head
            while (first != None)&(cnt < k):
                first = first.next
                cnt += 1
            if first == None: # 链表长度小于3
                return None
            second = head
            while first.next != None:
                first = first.next
                second = second.next
            return second

二进制中1的个数(剑指offer第十一题)

发表于 2017-09-01 | 分类于 数据结构

题目描述

输入一个整数，输出该数二进制表示中1的个数。其中负数用补码表示。

代码


    # -*- coding:utf-8 -*-
    class Solution:
        def NumberOf1(self, n):
            # write code here
            cnt = 0
            if n < 0:
                n = n&0xffffffff
            while n:
                cnt += 1
                n = n&(n-1)
            return cnt
