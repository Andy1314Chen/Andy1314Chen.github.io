LeetCode - 不同路径

发表于 2017-07-05 | 分类于 编程

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

问总共有多少条不同的路径？

例如，上图是一个7 x 3 的网格。有多少可能的路径？

说明：m 和 n 的值均不超过 100。

示例 1:

输入: m = 3, n = 2

输出: 3

解释:

从左上角开始，总共有 3 条路径可以到达右下角。

  1. 向右 -> 向右 -> 向下

  2. 向右 -> 向下 -> 向右

  3. 向下 -> 向右 -> 向右

示例 2:

输入: m = 7, n = 3

输出: 28

class Solution:
    def uniquePaths(self, m, n):
        """
        :type m: int
        :type n: int
        :rtype: int
        """
        # m是纵，n是横
        # 保持递归中间结果
        tmp = []
        for i in range(n+1):
            tmp.append([-1]*(m+1))
        # 递归调用
        return self.myAns(m, n, tmp)

    def myAns(self, m, n, tmp):
        # 出界认为是0
        if (m <= 0) | (n <= 0):
            return 0
        # 起点
        if (m == 1) & (n == 1):
            return 1
        # 已经递归过直接返回
        if tmp[n][m] != -1:
            return tmp[n][m]
        # 由结果往回走，左边部分，上边部分
        left = self.myAns(m-1, n, tmp)
        up = self.myAns(m, n-1, tmp)
        # 保持结果
        tmp[n][m] = left + up
        # 返回
        return tmp[n][m]

思路1(递归解法)

  1. 从结果回溯，某一点的路径总数等于其上一点，和其左一点的路径数和；
  2. m<= 0 || n<= 0 认为是边界，路径数为0；
  3. 保持递归过程的中间值；

思路2(递推解法)

  1. 与递归相反，从上到下，递推求解；

LeetCode - 寻找旋转排序数组中的最小值2

发表于 2017-07-05 | 分类于 编程

假设按照升序排序的数组在预先未知的某个点上进行了旋转。

( 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] )。

请找出其中最小的元素。

注意数组中可能存在重复的元素。

示例 1：

输入: [1,3,5]
输出: 1

示例 2：

输入: [2,2,2,0,1]
输出: 0

说明：

这道题是 寻找旋转排序数组中的最小值 的延伸题目。
允许重复会影响算法的时间复杂度吗？会如何影响，为什么？

class Solution(object):
def findMin(self, nums):
    """
    :type nums: List[int]
    :rtype: int
    """

    for i in range(1, len(nums)):
        if nums[i] < nums[i-1]:
            return nums[i]

    return nums[0]

def repeatedSubstringPattern(s):
    if len(s) < 2:
        return s
    length = len(s)
    for i in range(1, len(s)):
        sub = s[:i]
        # 包含多少个子串
        tmp = length//i
        # 剩余部分
        tmp2 = length%i
        if sub[:tmp2] == s[tmp*i:]:
            if tmp*sub == s[:tmp*i]:
                return sub
    return s

LeetCode - 解码方法

发表于 2017-07-05 | 分类于 编程

一条包含字母 A-Z 的消息通过以下方式进行了编码：

'A' -> 1
'B' -> 2
...
'Z' -> 26

给定一个只包含数字的非空字符串，请计算解码方法的总数。

示例 1:

输入: "12"
输出: 2
解释: 它可以解码为 "AB"（1 2）或者 "L"（12）。

示例 2:

输入: "226"
输出: 3
解释: 它可以解码为 "BZ" (2 26), "VF" (22 6), 或者 "BBF" (2 2 6) 。


class Solution:
    def numDecodings(self, s):
        """
        :type s: str
        :rtype: int
        """
        if len(s) == 0:
            return 0
        d = {}
        d[""] = 1
        return self.ways(s,d)

    # 记忆形递归
    def ways(self, s, d):
        # 已经计算过
        if s in d.keys():
            return d[s]
        # 首位为0不可解码
        if s[0] == '0':
            return 0
        # 长度为1
        if len(s) == 1:
            return 1
        # 去掉一位后的子问题
        first = self.ways(s[1:], d) # 去掉第一位
        tmp = int(s[:2]) # 看看前两位是否符合解码标准
        # 去掉两位后的子问题
        if (tmp > 9)&(tmp <= 26):
            first += self.ways(s[2:], d)
        d[s] = first
        return first
