LeetCode - 帕斯卡三角形

发表于 2017-07-05 | 分类于 编程

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。

在杨辉三角中，每个数是它左上方和右上方的数的和。

示例:

输入: 5
输出:
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]


    class Solution(object):
        def generate(self, numRows):
            """
            :type numRows: int
            :rtype: List[List[int]]
            """
            res = []
            for i in range(1, 1+numRows):
                res.append([1]*i)
            for i in range(2, numRows):
                for j in range(1, i):
                    res[i][j] = res[i-1][j] + res[i-1][j-1]
            return res

LeetCode - 第一个错误的版本

发表于 2017-07-05 | 分类于 编程

你是产品经理，目前正在带领一个团队开发新的产品。不幸的是，你的产品的最新版本没有通过质量检测。由于每个版本都是基于之前的版本开发的，所以错误的版本之后的所有版本都是错的。

假设你有 n 个版本 [1, 2, …, n]，你想找出导致之后所有版本出错的第一个错误的版本。

你可以通过调用 bool isBadVersion(version) 接口来判断版本号 version 是否在单元测试中出错。实现一个函数来查找第一个错误的版本。你应该尽量减少对调用 API 的次数。

示例:

给定 n = 5，并且 version = 4 是第一个错误的版本。

调用 isBadVersion(3) -> false
调用 isBadVersion(5) -> true
调用 isBadVersion(4) -> true

所以，4 是第一个错误的版本。 


    # The isBadVersion API is already defined for you.
    # @param version, an integer
    # @return a bool
    # def isBadVersion(version):

    class Solution(object):
        def firstBadVersion(self, n):
            """
            :type n: int
            :rtype: int
            """
            left = 1; right = n
            while left < right:
                mid = (left + right)//2
                if isBadVersion(mid) == True:
                    right = mid
                else:
                    left = mid + 1
            return left

LeetCode - 不同路径

发表于 2017-07-05 | 分类于 编程

一个机器人位于一个 m x n 网格的左上角 （起始点在下图中标记为“Start” ）。

机器人每次只能向下或者向右移动一步。机器人试图达到网格的右下角（在下图中标记为“Finish”）。

问总共有多少条不同的路径？

例如，上图是一个7 x 3 的网格。有多少可能的路径？

说明：m 和 n 的值均不超过 100。

示例 1:

输入: m = 3, n = 2

输出: 3

解释:

从左上角开始，总共有 3 条路径可以到达右下角。

  1. 向右 -> 向右 -> 向下

  2. 向右 -> 向下 -> 向右

  3. 向下 -> 向右 -> 向右

示例 2:

输入: m = 7, n = 3

输出: 28

class Solution:
    def uniquePaths(self, m, n):
        """
        :type m: int
        :type n: int
        :rtype: int
        """
        # m是纵，n是横
        # 保持递归中间结果
        tmp = []
        for i in range(n+1):
            tmp.append([-1]*(m+1))
        # 递归调用
        return self.myAns(m, n, tmp)

    def myAns(self, m, n, tmp):
        # 出界认为是0
        if (m <= 0) | (n <= 0):
            return 0
        # 起点
        if (m == 1) & (n == 1):
            return 1
        # 已经递归过直接返回
        if tmp[n][m] != -1:
            return tmp[n][m]
        # 由结果往回走，左边部分，上边部分
        left = self.myAns(m-1, n, tmp)
        up = self.myAns(m, n-1, tmp)
        # 保持结果
        tmp[n][m] = left + up
        # 返回
        return tmp[n][m]

思路1(递归解法)

  1. 从结果回溯，某一点的路径总数等于其上一点，和其左一点的路径数和；
  2. m<= 0 || n<= 0 认为是边界，路径数为0；
  3. 保持递归过程的中间值；

    class Solution:
        def uniquePaths(self, m, n):
            """
            :type m: int
            :type n: int
            :rtype: int
            """
            # 递推解法
            tmp = []
            for i in range(n+1):
                tmp.append([0]*(m+1))
            tmp[1][1] = 1

            for y in range(1, n+1):
                for x in range(1, m+1):
                    if (x==1) & (y==1):
                        continue
                    tmp[y][x] = tmp[y-1][x] + tmp[y][x-1]

            return tmp[n][m]

思路2(递推解法)

  1. 与递归相反，从上到下，递推求解；

LeetCode - 缺失数字

发表于 2017-07-05 | 分类于 编程

给定一个包含 0, 1, 2, …, n 中 n 个数的序列，找出 0 .. n 中没有出现在序列中的那个数。

示例 1:

输入: [3,0,1]
输出: 2

示例 2:

输入: [9,6,4,2,3,5,7,0,1]
输出: 8

说明:
你的算法应具有线性时间复杂度。你能否仅使用额外常数空间来实现?


    class Solution(object):
        def missingNumber(self, nums):
            """
            :type nums: List[int]
            :rtype: int
            """
            nums = sorted(nums)
            for i in range(len(nums)-1):
                if nums[i+1] > nums[i] + 1:
                    return i + 1
            if len(nums) == nums[-1]:
                return 0
            else: 
                return len(nums)

    输出: 5

    enention -> exention (将 'n' 替换为 'x')

            return self.myMethod(word1, word2, l1, l2, tmp)

        def myMethod(self, word1, word2, l1, l2, tmp):

LeetCode - 有效的括号

发表于 2017-07-05 | 分类于 编程

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串。

示例 1:

输入: "()"
输出: true

示例 2:

输入: "()[]{}"
输出: true

示例 3:

输入: "(]"
输出: false

示例 4:

输入: "([)]"
输出: false

示例 5:

输入: "{[]}"
输出: true


    class Solution(object):
        def isValid(self, s):
            """
            :type s: str
            :rtype: bool
            """
            if len(s)<1:
                return True

            d = {'(':')',
                '[':']',
                '{':'}'}
            # 当做栈
            res = []
            # 依次入栈
            for v in s:
                # 栈里有东西则进行比较，是否消除
                if len(res) > 0:
                    # 必须是左括号才能消除
                    if res[-1] in d.keys():
                        if d[res[-1]] == v:
                            res.pop()
                            continue
                res.append(v)
            if len(res) > 0:
                return False
            else:
                return True

class Solution(object):

LeetCode - 解码方法

发表于 2017-07-05 | 分类于 编程

一条包含字母 A-Z 的消息通过以下方式进行了编码：

'A' -> 1
'B' -> 2
...
'Z' -> 26

给定一个只包含数字的非空字符串，请计算解码方法的总数。

示例 1:

输入: "12"
输出: 2
解释: 它可以解码为 "AB"（1 2）或者 "L"（12）。

示例 2:

输入: "226"
输出: 3
解释: 它可以解码为 "BZ" (2 26), "VF" (22 6), 或者 "BBF" (2 2 6) 。


class Solution:
    def numDecodings(self, s):
        """
        :type s: str
        :rtype: int
        """
        if len(s) == 0:
            return 0
        d = {}
        d[""] = 1
        return self.ways(s,d)

    # 记忆形递归
    def ways(self, s, d):
        # 已经计算过
        if s in d.keys():
            return d[s]
        # 首位为0不可解码
        if s[0] == '0':
            return 0
        # 长度为1
        if len(s) == 1:
            return 1
        # 去掉一位后的子问题
        first = self.ways(s[1:], d) # 去掉第一位
        tmp = int(s[:2]) # 看看前两位是否符合解码标准
        # 去掉两位后的子问题
        if (tmp > 9)&(tmp <= 26):
            first += self.ways(s[2:], d)
        d[s] = first
        return first

LeetCode - 计数质数

发表于 2017-07-05 | 分类于 编程

统计所有小于非负整数 n 的质数的数量。

示例:

输入: 10
输出: 4
解释: 小于 10 的质数一共有 4 个, 它们是 2, 3, 5, 7 。

思路

  1. 从2开始遍历，将其平方，及平方+v的数，倍数全部置0
  2. 求和

    class Solution:
    def countPrimes(self, n):
        """
        :type n: int
        :rtype: int
        """
        tmp = [1]*n
        for v in range(2, n):
            if tmp[v-1]:
                # 该位置是质数
                # 将该质数的倍数全部标记
                for i in range(v*v, n, v):
                        tmp[i-1] = 0
        return sum(tmp[2:])
