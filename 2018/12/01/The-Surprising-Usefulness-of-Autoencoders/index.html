Autoencoder

自编码器是将输入复制到输出或学习辨识的方法。它有一个称为隐空间h的内部状态，用于表示输入。通常，此维度小于输入维度（称为欠完备）。自编码器由两部分组成：编码器f：X→H和解码器g：H→Y。

隐藏层维度应小于输入维度x。这样，h被迫学习输入空间的有用属性和最显着的特征。

Recurrent Autoencoder

对于时间序列数据，循环自编码器特别有用。唯一的区别是编码器和解码器被诸如LSTM之类的RNN取代。将RNN视为随时间变化的for循环，以保持状态。它可以展开到前馈网络中。

首先，将输入编码成不完备的隐向量h，然后由解码器对其进行解码。循环自编码器是序列到序列（seq2seq）架构的一种特殊情况，它在机器翻译中非常强大，将一种语言序列映射到另一种语言序列。

Task

复制两个异相初始化的正弦函数的张量。

输入的形状是形状的张量（batch_size，time_step，input_dim）。

  • batch_size 是训练的批次数。在单个样本上循环比在一批样本上应用张量操作要慢
  • time_step是RNN迭代的时间帧数。在本教程中，它是10，因为生成了10个点
  • input_dim是每个时间步的数据点数，这里我们有2个函数，所以这个数字是2

要处理股票数据，只需将input_dim轴替换为所需的数据点即可。

Generator

对于每个批次，生成2个正弦函数，每个函数有10个数据点。正弦函数的相位是随机的。

import random
def gen(batch_size):
    seq_length = 10

    batch_x = []
    batch_y = []
    for _ in range(batch_size):
        rand = random.random() * 2 * np.pi

        sig1 = np.sin(np.linspace(0.0 * np.pi + rand,
                                  3.0 * np.pi + rand, seq_length * 2))
        sig2 = np.cos(np.linspace(0.0 * np.pi + rand,
                                  3.0 * np.pi + rand, seq_length * 2))
        x1 = sig1[:seq_length]
        y1 = sig1[seq_length:]
        x2 = sig2[:seq_length]
        y2 = sig2[seq_length:]

        x_ = np.array([x1, x2])
        y_ = np.array([y1, y2])
        x_, y_ = x_.T, y_.T

        batch_x.append(x_)
        batch_y.append(y_)

    batch_x = np.array(batch_x)
    batch_y = np.array(batch_y)

    return batch_x, batch_x#batch_y

Model

目标是使用两个数字来表示正弦函数. 通常，我们使用φ∈R来表示三角函数的相位角。
让我们看看神经网络是否可以学习这个相位角。这里的计划是将输入正弦函数压缩为两个数字，然后将它们解码回来。

定义架构并让神经网络发挥作用。它是一个具有3层的模型，LSTM编码器将输入时间序列“编码”为固定长度的向量（在这种情况下为2）。RepeatVector将固定长度的向量重复10次，用作LSTM解码器的输入。对于解码器，我们可以用隐向量初始化隐藏状态（存储器）并使用时间t-1的输出作为时间t的输入，或者我们可以使用隐向量h作为每个时间步的输入。分别被称为条件和无条件解码器。

from keras.models import Sequential, Model
from keras.layers import LSTM, RepeatVector

batch_size = 100
X_train, _ = gen(batch_size)

m = Sequential()
m.add(LSTM(2, input_shape=(10, 2)))
m.add(RepeatVector(10))
m.add(LSTM(2, return_sequences=True))
print m.summary()
m.compile(loss='mse', optimizer='adam')
history = m.fit(X_train, X_train, nb_epoch=2000, batch_size=100)

由于这篇文章是该技术的演示，我们使用最小的模型，在这种情况下恰好是最好的。
“最佳”维度将是导致最高无损压缩的维度。在实践中，它主要是艺术而非科学。在生产中，有许多技巧可以加速训练并找到隐向量的正确容量(维数？)。主题包括：处理异步，非平稳时间序列的架构，预处理技术，如小波变换和FFT（Topics include: architectures for dealing with asynchronus, non-stationary time series, preprocessing techniques such as wavelet transforms and FFT.）

Visualization

该模型成功从20维压缩至2维。它从数据中学到了最显着的特征。然而，在这种情况下，模型是无用的，因为它已经成功地学习了g（f（x））= x。自动编码器设计为不完美的复制（？！为啥）。与所有神经网络一样，自动编码器利用数据集中在低维流形或流形某一集合的思想。目标是学习一个在这个流形的数据点上表现正确的函数，并在这个流形的各处(其他各处)展现出异常的行为。(因此autoencoder可以用来进行异常诊断)在这种情况下，我们发现了一个与20D空间相切的二维流形。这里我们讨论一些可视化技术。

下面我们将演示在用动画投影回20D时的流形图。理想情况下，这表明解码器的工作原理。

正如我们从上面的动画中看到的那样. 神经网络不使用相位角作为隐空间。相反，它在训练过程中使用由LSTM的梯度引起的这两个数字之间的复杂交互。As x→∞, the decoder g(h)→0. 这被理解为解码器不学习底层的三角函数，而仅仅是近似高维空间的流形。

Anomaly Detection

如你所见，自编码器是数据可视化和数据探索中非常强大的技术。但除了创建漂亮的图形之外，还有更深的应用。其中一个是异常检测。

监管机构可以通过构建无监督的深度学习算法来识别非法交易策略。

在数据挖掘中，异常检测（或异常值检测）是对不符合数据集中预期模式的观测值的识别。以无监督的方式训练神经网络的能力允许自动编码器在该区域中起作用。自动编码器为传统的信号重建和时间序列异常检测方法提供了非常强大的替代方案。

训练自动编码器在概念上很简单：

  1. 训练集训练Xtrain，with正则化
  2. 评估Xeval并确定自动编码器的容量
  3. 当我们在检查错误可视化后认为它足够好时，选择一个错误阈值（例如2σ）来确定数据点是否是异常。当然，由于我们正在处理时间序列，因此应该在滚动的基础上选择此阈值（滑动窗口）。

在金融时间序列（和其他系列）中，我们经常想要检测区域变化(突变？），即在价格突然变化之前知道。在我的实验中，这在检测冰山订单方面特别有效，冰山订单是为了隐藏实际订单数量而分成几个较小订单的单个大订单。使用普通机器学习方法检测冰山订单很困难，但在人工检查时很明显。

在下一部分中，我们将模拟异常检测如何工作。首先让我们绘制自编码器输出的均方误差。

    anomaly = np.argmax(mse)
    if mse[anomaly] > 0.4:
        plt.title("Anomaly detected in series {} at time step: {}".format(i, anomaly))

A Note on MSE

此外，假设金融系列被归一化为零中心信号，MSE将是一种适应不良的措施。因为如果异常值接近0，则MSE可能小于原始信号的MSE（以0为中心）。因此，必须通过向MSE添加移动平均值来设计更复杂的损失函数。这样，我们可以根据与滚动平均值的偏差来判断。

A Note on Online Training / Inference

与所有信号处理神经网络一样，该算法应该能够在滚动的基础上运行。该算法只需要来自滑动窗口的一段数据。

当引入注意力机制时，在线训练（部分观察到的输入）变得更加复杂。使用记忆机制可以实现在线和线性时间注意对齐。我可能会在未来的帖子中介绍金融数据的关注力机制。

Pattern Search

给出一个示例交易模式，快速识别多维数据系列中的类似模式。

实现：

  • 在序列数据集上训练堆叠的循环自动编码器
  • 对查询序列模式Xt-1，Xt，Xt + 1，… 进行编码以获得HX
  • 然后对于所有历史数据，使用编码器产生固定长度矢量H = {Ha，Hb，Hc，…}
  • 对于每个t∈H，找到argmintdist（Ht，HX）

它基本上是一个模糊散列函数。我没有尝试过，因为计算所有序列的哈希值需要严格的计算才能供大型机构玩家使用。

Stacked Autoencoders

这是介绍堆叠自动编码器如何工作的好地方。

如上所述，简单的循环自动编码器有3层：编码器LSTM层，隐藏层和解码器LSTM层。堆叠自动编码器通过堆叠多个单层自动编码器构成。第一个单层自动编码器将输入映射到第一个隐向量。在训练第一自动编码器之后，我们丢弃第一解码器层，然后由第二自动编码器替换，第二自动编码器具有较小的隐向量维度。重复此过程并通过反复试验确定正确的深度和大小（如果您知道更好的方法，请告诉我。）堆叠自动编码器的深度使功能不变，并允许提取的功能更抽象。

A Note on Supervised Learning

这篇文章仅涉及自动编码器的无监督使用。变分复现自动编码器可以“去噪”人为损坏的数据输入。我从来没有运气，所以我不推荐在金融数据上使用。

金融数据非常嘈杂。但是，当训练容量较小时，自动编码器具有内置的去噪能力，因为它们只能学习最显着的功能。
