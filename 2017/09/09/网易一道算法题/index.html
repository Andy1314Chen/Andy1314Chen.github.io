网易校招一道算法题

发表于 2017-09-09 | 分类于 编程

题目是：

小易正在玩一款新出的射击游戏，这个射击游戏在一个二维平面进行
，小易在坐标原点(0,0)，平面上有n只怪物，每个怪物有所在的坐标
(x[i],y[i]).小易进行一次射击会把x轴和y轴上(包括坐标原点)的怪
物一次性消灭。小易是这个游戏的VIP玩家，他拥有两项特权操作。

1、让平面内的所有怪物同时向任意同一方向移动任意同一距离。

2、让平面内的所有怪物同时对于小易(0,0)旋转任意同一角度。

小易要进行一次射击，小易在进行射击前，可以使用这两种特权

操作任意次。

小易想知道在他射击的时候最多可以同时消灭多少只怪物。

看到题目的时候，首先想的是根据两项特权暴力遍历。后来理解题意，发现怪物数n<=50, 两项特权的意思其实是可以改变坐标系的位置和方向，那么在改变的过程中坐标系x轴，y轴出现怪物次数最多。

想到这一点之后，便开始动手编程。思路是，首先选择一个坐标，让其他怪物坐标减去这个坐标(相当于将坐标系移动到该坐标了)，然后计算相对选定怪物的角度[-180,+180]，再将同一角度(45和135属于一类)的分类。按角度类别再次遍历，寻找某一角度和其角度垂直的角度的个数，再加上原点1个就是当前怪物为原点的最大消灭数。

真实的讲，想到这一点的时候还是很激动的~ 但是后来用C语言怎么也实现不了。原因在于无法快速找到同一角度的分类，一个数组里不重复的数有多少哪些，分别有多少个？在这里卡了好久。。。

后来看到牛客网上有人公布答案，解题思路类似，但是他是用斜率来表示的，更高明一些。同一条直线上斜率相同，垂直直线斜率乘积是-1；

C++代码如下：

作者：雨baby
链接：https://www.nowcoder.com/discuss/39219?type=0&order=0&pos=6&page=2
来源：牛客网

#include <bits/stdc++.h>

using namespace std;

const int maxn = 50 + 5;

int x[maxn], y[maxn];
int n;
int solve() {
    if(n <= 2) return n;
    int res = 1;
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < n; j++) {
            if(i != j) {
                int dx1 = x[j] - x[i];
                int dy1 = y[j] - y[i];
                for(int k = 0; k < n; k++) {
                    int cnt = 0;
                    if(i != k && j != k) {
                        for(int r = 0; r < n; r++) {
                            int dx2 = x[r] - x[i];
                            int dy2 = y[r] - y[i];
                            if(dy1 * dx2 == dy2 * dx1) {
                                cnt++;
                            } else {
                                dx2 = x[r] - x[k];
                                dy2 = y[r] - y[k];
                                if(dy1 * dy2 == -dx2 * dx1) {
                                    cnt++;
                                }
                            }
                        }
                    }
                    res = max(res, cnt);
                }
            }
        }
    }
    return res;
}
int main() {
    cin >> n;
    for(int i = 0; i < n; i++) cin >> x[i];
    for(int i = 0; i < n; i++) cin >> y[i];
    cout << solve() << endl;
}
