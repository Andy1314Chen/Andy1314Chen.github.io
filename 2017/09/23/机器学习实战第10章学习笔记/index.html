K-均值聚类算法

K-均值是发现给定数据集的k个簇的算法。簇个数k是用户给定的，每个簇通过其质心(Centroid)，即簇中所有点的中心来描述。

K-均值算法的工作流程是：首先随机确定k个初始点作为质心，然后将数据集中的每个点分配到一个簇中，具体而言，为每个点找到距其最近的质心（Centroid),并将其分配给该质心所对应的簇。这一步之后，每个簇的质心更新为该簇所有点的平均值。

K-均值聚类

  • 优点：容易实现
  • 缺点：可能收敛到局部最小值，在大规模数据集上收敛较慢。
  • 适用数据类型：数值型数据。

算法伪代码如下：

创建k个点作为起始质心（经常是随机选择）
当任意一个点的簇分配结果发生改变时
    对数据集中的每个数据点
        对每个Centroid
            计算Centroid到数据点之间的距离
        将数据点分配到距其最近的簇
    对每一个簇，计算簇中所有点的均值并将均值作为Centroid

K-均值聚类的一般流程

  1. 收集数据：使用任意方法
  2. 准备数据：需要数值型数据来计算距离，也可以将标称型数据映射为二值型数据再用于距离计算
  3. 分析数据：使用任意方法
  4. 训练算法：不适应于无监督学习，即无监督学习没有训练过程
  5. 测试算法：应用聚类算法、观察结果。可以使用量化的误差指标如误差平方和来评价算法的结果。
  6. 使用算法：可以用于所希望的任何应用。通常情况下，簇Centroid可以代表整个簇的数据来做出决策。

K-均值聚类支持函数：

import numpy as np

#K-均值聚类支持函数
def loadDataSet(fileName):
    dataMat=[]
    fr=open(fileName)
    for line in fr.readlines():
        curLine=line.strip().split('\t')
        fltLine=list(map(float,curLine))
        dataMat.append(fltLine)
    return dataMat

def distEclud(vecA,vecB):
    return np.sqrt(np.sum(np.power(vecA-vecB,2)))

def randCent(dataSet,k):
    n=np.shape(dataSet)[1]
    centroids=np.mat(np.zeros((k,n)))
    for j in range(n):
        minJ=np.min(dataSet[:,j])
        rangeJ=float(np.max(dataSet[:,j])-minJ)
        #确保随机质心在数据的边界之内
        centroids[:,j]=minJ+rangeJ*np.random.rand(k,1)
    return centroids

上述三个函数功能分别是，读取数据、计算距离、随机初始化k个Centroid.

#K-均值聚类算法
"""
算法思路：
    初始化分配k个质心；
    计算样本距离，分配样本；
    重新计算k个质心；
    重复以上过程，直到所有数据点的分配结果不再变化
"""

def kMeans(dataSet,k,distMeas=distEclud,createCent=randCent):
    m=np.shape(dataSet)[0]
    #存储每个样本点的分配结果，第一列记录簇索引值，第二列存储误差
    clusterAssment=np.mat(np.zeros((m,2)))
    #随机产生K个质心
    centroids=createCent(dataSet,k)
    clusterChanged=True
    while clusterChanged:
        clusterChanged=False
        for i in range(m):
            minDist=np.inf;minIndex=-1
            for j in range(k):
                #计算样本距离
                distJI=distMeas(centroids[j,:],dataSet[i,:])
                if distJI<minDist:
                    #找到与Xi最近的质心
                    minDist=distJI;minIndex=j
            if clusterAssment[i,0]!=minIndex:
                clusterChanged=True
            clusterAssment[i,:]=minIndex,minDist**2
        #print(centroids)
        #更新k个质心
        for cent in range(k):
            ptsInClust=dataSet[np.nonzero(clusterAssment[:,0].A==cent)[0]]
            centroids[cent,:]=np.mean(ptsInClust,axis=0)
    return centroids,clusterAssment

在K-均值聚类中簇的数目k是一个用户预先定义的参数，那么用户如何才能知道k的选择是否正确？如何才能知道生成的簇比较好呢？在包含簇分配结果的矩阵中保持着每个点的误差，即该点到簇Centroid的距离平方值。

K-均值算法可能会收敛到局部最小值，而非全局最小值。一种用于度量聚类效果的指标是SSE(Sum of Squared Error,误差平方和）。SSE值越小表示数据点越接近于它们的Centroid,聚类效果也越好。

二分K-均值算法

为克服K-均值算法收敛于局部最小值的问题，有人提出了二分K-均值（bisecting K-means)的算法。基本思路是：首先将所有点作为一个簇，然后将该簇一分为二，之后选择其中一个簇继续进行划分，选择哪一个簇进行划分取决于对其划分是否可以最大程度降低SSE的值。该过程不断重复，直到得到用户指定的簇数目为止。

#二分K-均值算法
“””
该算法首先将所有点作为一个簇，然后将该簇一分为二。之后选择其中一个簇继续进行划分，
选择哪一个进行划分取决于对其划分是否可以最大程度降低SSE的值。上述基于SSE的划分过程
不断重复，直到得到用户指定的簇数目为止。

    二分K-均值算法的伪代码：
    将所有点看成一个簇
    当簇数目小于k时
    对于每一个簇
        计算总误差
        在给定的簇上面进行K-均值聚类（k=2）
        计算将该簇一分为二之后的总误差
    选择使得误差最小的那个簇进行划分操作
"""
#二分K-均值聚类算法
def biKmeans(dataSet,k,distMeas=distEclud):
    m=np.shape(dataSet)[0]
    #存储数据集中每个点的簇分配结果及平方误差
    clusterAssment=np.mat(np.zeros((m,2)))
    #计算该数据集的均值，并以此作为数据集的初始簇的质心
    centroid0=np.mean(dataSet,axis=0).tolist()[0]
    centList=[centroid0]
    for j in range(m):
        #计算每个样本的距离
        clusterAssment[j,1]=distMeas(np.mat(centroid0),dataSet[j,:])**2
    #小于k个聚类
    while(len(centList)<k):
        lowestSSE=np.inf
        #遍历所有簇来决定最佳簇进行划分
        for i in range(len(centList)):
            #选择出第i个聚类
            ptsInCurrCluster=\
                dataSet[np.nonzero(clusterAssment[:,0].A==i)[0],:]
            #对其进行划分
            centroidMat,splitClustAss=\
                kMeans(ptsInCurrCluster,2,distMeas)
            #划分后的SSE
            sseSplit=np.sum(splitClustAss[:,1])
            #未划分的SSE
            sseNotSplit=\
                np.sum(clusterAssment[np.nonzero(clusterAssment[:,0].A!=i)[0],1])
            print("sseSplit, and notSplit: ",sseSplit,sseNotSplit)
            if (sseSplit+sseNotSplit)<lowestSSE:
                bestCentToSplit=i
                bestNewCents=centroidMat
                bestClustAss=splitClustAss.copy()
                lowestSSE=sseSplit+sseNotSplit
        #将簇编号修改为新编号
        bestClustAss[np.nonzero(bestClustAss[:,0].A==1)[0],0]=\
            len(centList)
        bestClustAss[np.nonzero(bestClustAss[:,0].A==0)[0],0]=\
            bestCentToSplit
        print('the bestCentToSplit is: ',bestCentToSplit)
        print('the len of bestClustAss is: ',len(bestClustAss))
        centList[bestCentToSplit]=bestNewCents[0,:]
        centList.append((bestNewCents[1,:]))
        clusterAssment[np.nonzero(clusterAssment[:,0].A==\
                                  bestCentToSplit)[0],:]=bestClustAss
    return centList,clusterAssment

def massPlaceFind(fileName):
    fw=open('places.txt','w')

#对地理坐标进行聚类  （使用球面余弦定理来计算两个经纬度之间的距离）
#返回地球表面两点之间的距离
def distSLC(vecA,vecB):
    a=np.sin(vecA[0,1]*np.pi/180)*np.sin(vecB[0,1]*np.pi/180)
    b=np.cos(vecA[0,1]*np.pi/180)*np.cos(vecB[0,1]*np.pi/180)*\
        np.cos(np.pi*(vecB[0,0]-vecA[0,0])/180)
    return np.array(np.arccos(a+b)*6371.0)

小结

聚类是一种无监督的学习方法。所谓无监督学习是指事先并不知道要寻找的内容，即没有目标变量。聚类将数据点归到多个簇中，其中相似数据点处于同一簇，而不相似数据点处于不同簇中。这一章主要学习k-均值聚类算法和二分K-均值聚类算法。
