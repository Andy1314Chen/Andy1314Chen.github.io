How to Make Out-of-sample Forecast with ARIMA in Python3

发表于 2017-10-08 | 分类于 Python ， 算法

最近在看时间序列分析这一块，因为不会用R语言（R语言做时间序列分析很是方便！），只能用Python吭哧吭哧的搞，搞了好多天，在网上找了一些资料，比较乱，基本上抄来抄去，到最后也没有搞出个眉目。

今天看到一篇老外写的博客，国内没发现有类似的，打算翻译一下，顺带加深Python下ARIMA的使用。

没错，图片也是从人家博客里copy的

作者说的一点没错，

Making out-of-sample forecasts can be confusing when getting started with time series data.

的确困扰了我好久，总看到in-of-sample forecast & out-of-sampl e forecast，但也没人去解释。

in-of-sample forecast是指利用建模的那些时间序列数据去做预测，没错就是这样，和其他机器学习模型不太一样。而out-of-sample forecast是利用训练集训练好的数据去预测未来的数据。

python下里的时间序列库最常见的是statsmodels,最常用的模型是ARIMA.

这篇博客里，老外主要是以一个简单数据集为例，讲解了一下3个东西（ARIMA模型的）

  • How to make a one-step out-of-sample forecast
  • How to make a multi-step out-of-sample forecast
  • The difference between the forecast() & predict()

数据集下载

数据集是澳大利亚，墨尔本从1981-1990年10年间每日最低温度的一个数据集。

下载地址

这个数据集中有一些“？”需要去除。

分割数据集

将数据集分为两个部分，最后7天的数据分出来作为验证集，前面的一部分作为训练集。

建模

从下面的图中可以看到，该时间序列有明显的季节特性，通过简单的差分可以去掉。季节特性是以年为周期的，365天。其实，差分之后应该还要进行统计检验的，确保差分后的时间序列为平稳序列。

使用ARIMA模型，通过网格搜索法找到最佳参数（7,0,1）。

One-Step Out-of-Sample Forecast

ARIMA模型提供了两种方法

  1. Forecast Function
  2. Predict Function

对out-of-sample预测结果都是一样的，但是predict()可以in-of-sample预测，好像更为先进一些，但是好像也没太大用处，只是验证模型。

预测之后，还需要对预测值进行还原处理，因为之前对数据进行了差分处理。

Multi-Step Out-of-Sample Forecast

同单步预测差不多，都是两个函数可以用，但是还原时要注意。

#create a differenced series
def difference(dataset,interval=1):
    diff=list()
    for i in range(interval,len(dataset)):
        value=dataset[i]-dataset[i-interval]
        diff.append(value)
    return np.array(diff)

#invert differenced value
def inverse_difference(history,yhat,interval=1):
    return yhat+history[-interval]

from statsmodels.tsa.arima_model import ARIMA

One-Step Out-of-Sample Forecast

ARIMA模型进行一步预测是很好用的。
在这里，一步预测的是1990-12-25日的最低气温。

statsmodel库提供了forecast() function进行预测。默认，这个函数就会进行一步预测（out-of-sample forecast）.forecast()返回值包含

the forecast value,the standard error of the forecast, and the confidence interval information

#one-step out-of-sample forecast
forecast=model_fit.forecast()[0]

#前面为了消除时间序列的季节特性，进行了差分，这里得到的预测值必须要还原
forecast=inverse_difference(X,forecast,days_in_year)

print("Forecast: %f" % forecast)

statsmodel库同样提供了predict() function函数用于预测，可以进行任意in-of-sample & out-of-sample的预测。但是predict()需要提供一个start和end.

#one-step out of sample forecast
start_index=len(differenced)
end_index=len(differenced)
forecast=model_fit.predict(start=start_index,end=end_index)
#forecast=model_fit.predict('1990-12-25','1990-12-25')

#invert the differenced forecast to something usable
forecast=inverse_difference(X,forecast,days_in_year)
print("Forecast: %f" % forecast)

Forecast: 14.861660

可以看到predict()预测结果和forecast()是一样的，不过predict()即可以in-of-sample也可以out-of-sample.

Multi-Step Out-of-Sample Forecast

同样可以使用predict()&forecast()函数进行多步预测。

#multi-step out-of-sample forecast
forecast=model_fit.forecast(steps=7)[0]
#invert the differenced forecast to something usable
history=[x for x in X]
day=1
result=[]
for yhat in forecast:
    inverted=inverse_difference(history,yhat,days_in_year)
    print("Day %d: %f" % (day,inverted))
    history.append(inverted)
    result.append(inverted)
    day+=1
